说明：

利用业余时间写的一个Excel插件，除了能够生成被Testlink识别的XML文档外，该插件可帮助Tester检查当前有多少需求被覆盖，已经当前Test Case覆盖了
哪些需求，Test case的覆盖率和自动化率，其检查结果会以报表的方式显示在Excel中。目前包含的功能有：

Feature List:

- New Test Suite
- Export Current Test Suite
- New Requirement Collection
- Export Current Requirement collection
- Analyze All Test Suites;
- Analyze All Requirements;
